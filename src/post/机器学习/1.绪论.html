绪论

重点掌握：基本概念、背景、典型应用

一.基本概念

（一）、课程背景 1. 人工智能是软件发展的新方向之一 2. 机器学习是人工智能的最核心技术 2. 机器学习体系庞大 - 前沿、历史 - 理论基础 - 技术实现 - 应用实现

（二）、关键概念的内涵与外延 1. 人工智能（科学、技术） 研究使计算机来模拟人的某些思维过程和智能行为（如学习（记忆和复现）、推理、思考、规划等 2. 中国新一代人工智能（产业、技术） - 基础：移动互联网、大数据、超级计算、传感网、脑科学等新理论新技术 - 影响经济、社会、国际竞争、人类社会的不确定性 - 芯片、硬件、云、算法、计算加速、软件平台、API 3. 机器学习（科学、技术） - 面向特定任务的能够自我学习的程序，从经验数据中学习潜在规律，进而成为解决经验型问题的一种方法 - 新的编程范式

（三）实例 1. Handwriting Recognition + 用途：邮件自动分拣、手写支票自动识别 + 常识：如果积累了足够多的手写字符，那么新来的手写字符的识别是可以靠积累的历史字符推断出来的 + 总结：从过去到现在手写的字符是有统计规律的，可以在历史中总结出来这个经验用于推断即将到来的手写字符 + 存在的问题： - 用何种形式表示经验 - 如何从历史数据中提炼经验 + 解决手段 - 使用贝叶斯分类器 - 使用概率表示数据中呈现的统计规律 - 使用极大后验假设判别（MAP） 2. 汽车自动驾驶 + 输入：利用激光传感器数据，摄像机视频数据，车辆定位仪，避碰雷达的距离数据等。 + 输出：油门/刹车，方向盘转动角度 3. 国际象棋（AlphaGo） + Policy NetWork：outputs a probability distribution or over legal moves + Value NetWork：outputs a scalar value that predicts the expected outcome in position + SLPN：使用人类棋局训练 + RLPN：和自己对弈大量棋局，使用强化学习进一步改善它 4. 自动问答（基于自然语言处理和知识工程） + ChatGPT

总结 + 找出模型，可以是概率、神经网络、线性函数等等 + 预测：利用规律预测新样本的结果 - f 是样本潜在规律 - 利用x学习得到f，计算得到预测值y + 训练：从历史经验中学习规律 - 给定一组样本（历史经验），学习得到 - f 可以是等等

二、机器学习

（一）、什么是机器学习 A computer program is said to learn from experience E with respect to some class of tasks T and performance measure P, if its performance at tasks in T, as measured by P, improves with experience E

（二）、按输入输出分 1. 分类 将事物按一定标准分成一些类别，输出是有限集合，例如（yes、no） 2. 回归 由过去、现在的数据计算出未来状态，输出是连续值 3. 聚类 没有类别的标准，按事物间的相似性划分成一些类别 + 聚类是通过样本集合划分来获取对数据分布的认识 + 划分依据：相近（人以类聚、物以群分） + 也被称为无监督学习 - 对于无监督学习，只有{xi}，没有可供训练的样本标签y。 - 相对于有监督学习（如分类和回归），需要训练样本{(xi,yi)}
